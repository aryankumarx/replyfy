## 🚀 Quick Start (FREE Version)

### What to Install:
1. **Node.js** - https://nodejs.org/ (FREE)
2. **VS Code** - https://code.visualstudio.com/ (FREE)
3. **Android Studio** - https://developer.android.com/studio (FREE)
4. **Gemini API Key** - https://makersuite.google.com/app/apikey (FREE)

**Total Cost: $0**

### Setup Steps:
```bash
# 1. Get Gemini API Key (2 min)
Go to: https://makersuite.google.com/app/apikey

# 2. Setup backend
cd backend
cp package-gemini-free.json package.json
npm install
cp .env.example .env
# Add your GEMINI_API_KEY to .env

# 3. Setup frontend
cd ../frontend
npm install

# 4. Run it!
# Terminal 1:
cd backend && npm run dev

# Terminal 2:
cd frontend && npm start

# Terminal 3:
cd frontend && npx react-native run-android
```
