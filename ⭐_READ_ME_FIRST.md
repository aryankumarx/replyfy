## ⚡ Super Quick Start (If Experienced)

**Already have Node.js and Android Studio?**

```bash
# Get Gemini key: https://makersuite.google.com/app/apikey

cd backend
cp package-gemini-free.json package.json
npm install
cp .env.example .env
# Add GEMINI_API_KEY to .env
npm run dev

# New terminal
cd frontend
npm install
npm start

# New terminal
cd frontend
npx react-native run-android
```

**Done!**
