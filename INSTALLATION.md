# Replyfy - Installation Guide

## Install
1. Download `Replyfy.apk` from the [Releases page](https://github.com/aryankumarx/replyfy/releases/latest).
2. Open the APK and follow the install prompt.
3. If Play Protect appears, tap **More details** → **Install anyway**.
4. If installation is still blocked, temporarily disable **Scan apps with Play Protect**, install the APK, then re-enable it.

## Permissions
- **Display over other apps**: enables the floating bubble.
- **Accessibility Service**: detects chat apps.
- **Internet**: connects to the AI backend.

## First-time setup
1. Open Replyfy.
2. Grant overlay permission.
3. Enable the Accessibility Service.
4. Copy a message from WhatsApp, Telegram, or SMS.
5. Tap the bubble to generate replies.

## Safety
Replyfy is open source and does not collect user data. Messages are processed locally, and sensitive data is blocked before sending.

## Troubleshooting
| Problem | Fix |
|---|---|
| Bubble doesn't appear | Check overlay permission in Settings → Apps → Replyfy. |
| No suggestions | Check your internet connection. |
| Play Protect blocks installation | Temporarily disable it, install the APK, then re-enable it. |
