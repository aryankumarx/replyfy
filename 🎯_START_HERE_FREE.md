### Something Not Working?

2. **Search:** The documentation (Ctrl+F)
3. **Read:** Error messages carefully
4. **Ask:** Create GitHub issue

### Common Issues Quick Links:

| Issue | Solution |
|-------|----------|
| Node not found | Restart computer after installing |
| Port 3000 in use | Kill process: `lsof -ti:3000 \| xargs kill -9` |
| Emulator won't start | Recreate device in Android Studio |
| Module not found | Run `npm install` in that folder |
| App can't connect | Use `http://10.0.2.2:3000/api` for emulator |
